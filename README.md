# ChatConnect - A Simple Android Messaging App (Java)

ChatConnect is a basic chat/messaging app built with Android Jva that demonstrates learned key development concepts
i.e RecyclerView, Adapters, Fragments, passing data between activities, and Services.
It will include real-time chat with Firebase, local storage with Room Database, and push notifications.

Features (WIP)
  - Chat history List (RecyclerView + Adapter) – history conversations.
  - Chat Screen (Passing Data Between Activities) – Shows user messages dynamically.
